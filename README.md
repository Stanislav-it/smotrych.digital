Aktualizacja:

- logo użyte w headerze, footerze i faviconie;
- po hero/headerze dodana płynna sekcja portfolio z trzema telefonami z dostarczonego ZIP-a;

## Start

```bash
pip install -r requirements.txt
python app.py
```

Strona działa na:

```text
http://127.0.0.1:5001
```
